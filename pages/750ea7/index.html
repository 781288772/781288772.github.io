解决父组件重新赋值子组件数据缓存问题

# 父组件重新请求网络数据并赋值给子组件，子组件渲染遗留有旧数据

前些天遇到这么一个问题：在一个个人信息页面上修改个人信息之后返回列表渲染页面， 惊奇地发现列表的数据包含了更新信息之前的数据，也就是数据条数是原来的两倍，经过 漫长的查找原因之后，找到了原因，组件没有刷新。找到问题所在的话，那么就可以着手 来解决问题了。

# 解决方案：使用v-if使子组件重新渲染，让组件强制刷新。

//子组件
<children v-if="isShow"></children>


//重置子组件的数据
clear(){
    this.isShow = false;
    this.$nextTick(()=>{
    this.isShow = true;
    })
 }
